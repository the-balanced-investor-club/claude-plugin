Load the `sector-overview` skill and create an industry landscape report covering market sizing, competitive dynamics, and the open questions the sector has not settled.

Report the distribution, not just the average — the weighted and the simple return usually disagree, and the gap is the finding.

If a sector is provided, use it. Otherwise ask the user which industry to cover.
