---
description: Build a comparable company analysis — peers, multiples, and the spread between them
argument-hint: "[company name or ticker]"
---

Load the `comps-analysis` skill and build the peer set: 4–6 genuinely comparable companies, their operating metrics and their valuation multiples, with the statistical summary — max, 75th, median, 25th, min.

Report the **quartiles, not the average**. An average across six companies is one outlier away from meaningless, and the spread is what tells the reader whether this is a peer set at all.

Trading below the peer median is a fact about the multiple, not a fact about the company. **Why it trades there is the question the table exists to raise — never to answer.**

If a ticker is provided, use it. Otherwise ask which company.
