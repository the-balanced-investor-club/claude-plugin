Load the `ownership-context` skill.

Check insider "Buy" rows against the market price — below-market means an option strike, not a purchase. Report institutional holders that increased against those that decreased, never the ownership percentage alone, and state how stale each 13F is.

If a ticker is provided, use it. Otherwise ask which company.
